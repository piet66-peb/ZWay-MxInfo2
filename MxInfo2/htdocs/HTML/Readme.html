MxInfo2 Tool Collection

This package contains a collection of useful analysis tools. They are designed to run in a browser on a desktop PC. The module does nothing but download the software.

Tools Overview

Tool Purpose
BatteryStates Displays current and historical battery charge levels.
Communication Analysis of Z-Wave communication from incoming and outgoing packets (PacketLog and Zniffer), statistics per hour, node, command..., drawing of historical sensor values
ConfigJson Contents of config.json, ability to remove old, not any more existing devices
CronTasks Overview of all Z-Way cron tasks, table of next fires
DataRates Statistics about used Z-Wave data rates (speed)
Dependencies Devices and module instances and their dependencies (graphic representation), classification of devices
DeviceParameters Display and change device parameters. Comparing the parameterization of 2 devices of the same type. The Parameters are shown side by side and differences are marked. Parameters can be changed.
Device parameterizations can be printed into pdf files for documentation purpose.
Devices A list of all existing virtual devices.
InstanceCompare Intended to check the configuration of 2 module instances. The instances are shown side by side for comparison.
Instance parameterizations can be printed into pdf files for documentation purpose.
Instances A list of all existing module instances.
Routes Theoretical possible routes for Z-Wave packets, used and failed routes.
Routing Routing statistics, set/delete priority routes.
Security List of secure included devices.
ZwayConfiguration All device data in Z-Way system. Some flags can be changed.

Login as administrator is necessary first.
